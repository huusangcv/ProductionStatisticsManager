Before finishing any task:

✓ Reuse existing components
✓ No duplicate code
✓ Route updated if required
✓ Sidebar menu updated if required
✓ Theme consistency preserved
✓ IPC channel added if backend changes
✓ Database migration updated if schema changes
✓ Build succeeds
✓ ESLint passes
