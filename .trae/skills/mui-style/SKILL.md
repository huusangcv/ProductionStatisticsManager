# Material UI Style Guide

Always use Material UI.

Never use Bootstrap.

Never use Tailwind.

Preferred components:

Box

Stack

Paper

Card

Container

Typography

Button

TextField

DataGrid

Dialog

Drawer

Tabs

Menu

Avatar

Icons

Use theme values instead of hardcoded values.

Use

theme.spacing()

theme.palette

theme.shape.borderRadius

Never hardcode colors.

4

12px

Soft only.

Enterprise style.

No gradients.

No glassmorphism.

No neumorphism.
